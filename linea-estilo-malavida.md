# Linea de estilo visual para Charanga Malavida

## 1. Resumen visual del logo

El logo de **Charanga Malavida** funciona como un sello circular de alto impacto. La composicion combina tres elementos principales:

- Un anillo exterior negro muy dominante.
- Texto blanco en mayusculas, condensado y curvado alrededor del circulo.
- Una mascota central: corazon rosa con cuernos, cara cartoon, guantes y zapatillas.

La lectura visual es inmediata: contraste extremo, humor, energia de calle y actitud gamberra pero cercana. No es una marca discreta ni institucional; es una identidad pensada para verse en camisetas, bombos, pegatinas, redes y escenarios festivos.

La web debe trasladar esa sensacion: directa, grafica, reconocible y divertida. Debe parecer la web de una charanga con personalidad, no una plantilla generica de eventos.

## 2. Personalidad de marca

La marca transmite:

- **Fiesta**: el logo pide movimiento, musica y ambiente popular.
- **Gamberrismo amable**: el corazon con cuernos tiene punto travieso, pero no agresivo.
- **Cercania**: la mascota sonrie y saluda; invita a entrar.
- **Calle**: el sello circular y el trazo grueso funcionan muy bien para pegatina, parche o cartel.
- **Retro cartoon**: hay ecos de mascota clasica, comic, vinilo, sticker y merchandising.
- **Alto volumen visual**: negro, blanco y rosa se leen desde lejos.

La web deberia hablar con frases cortas, llamadas a la accion claras y una estetica con mucho contraste. El tono visual puede ser potente sin perder orden.

## 3. Paleta de color recomendada

| Negro principal | Negro logo | `#000000` |
| Blanco principal | Blanco logo | `#FFFFFF` |
| Rosa Malavida | Corazon central | `#F66AA6` |

### Colores secundarios

| Uso | Color | HEX aproximado |
| --- | --- | --- |
| Rosa oscuro | Hover, sombra, enfasis | `#D93D82` |
| Rosa claro | Fondos suaves | `#FFD3E6` |

### Proporcion recomendada

- Negro: 35-45%.
- Blanco o blanco roto: 30-40%.
- Rosa Malavida: 15-25%.
- Grises y rosas suaves: solo como apoyo.

La web no debe convertirse en una web rosa entera. El rosa debe ser memorable porque aparece en lugares estrategicos: CTA, fondos de impacto, detalles graficos, estados activos y elementos destacados.

## 4. Reglas de contraste

El logo se basa en contraste maximo. La web debe mantener esa energia:

- Texto negro sobre blanco o blanco roto para lectura larga.
- Texto blanco sobre negro para hero, footer, bloques de impacto y llamadas fuertes.
- Rosa sobre negro funciona muy bien como acento.
- Negro sobre rosa funciona bien para botones, badges y titulares cortos.
- Evitar texto blanco sobre rosa si el rosa se usa muy claro.
- Evitar grises suaves para informacion importante.

Reglas practicas:

- Los titulares principales deben usar negro o blanco, no grises.
- Los botones principales deben tener contraste fuerte.
- Las tarjetas deben tener borde negro visible.
- Las sombras pueden ser duras y graficas, no difuminadas.
- Las secciones deben alternar masas claras y oscuras para mantener ritmo.

## 5. Recomendaciones tipograficas

El logo usa una tipografia condensada, muy alta, en mayusculas, con aire de cartel y sello. Para la web:

### Titulares

Usar una fuente display condensada, gruesa y con caracter. Opciones recomendadas:

- **Bebas Neue**
- **Anton**
- **Oswald**
- **Archivo Black**
- **League Gothic**

Uso:

- H1 y H2 en mayusculas o casi mayusculas.
- Titulares cortos, con mucho peso.
- No usar letter spacing negativo.
- Puede funcionar un ligero espaciado positivo en etiquetas pequenas.

### Texto de lectura

Usar una sans limpia y legible:

- **Inter**
- **Montserrat**
- **Nunito Sans**
- **Arial/Helvetica** si se quiere mantener simple.

Uso:

- Parrafos con buen interlineado.
- No abusar de mayusculas en textos largos.
- Botones y badges en peso alto.

### Jerarquia recomendada

- Hero: display condensada, muy grande.
- Secciones: titulares contundentes.
- Tarjetas: titulo fuerte + descripcion clara.
- Formularios: labels legibles, campos simples.

## 6. Estilo de botones, tarjetas, fondos y secciones

### Botones

Los botones deben parecer piezas graficas, no controles corporativos suaves.

Recomendaciones:

- Bordes negros de 2-3px.
- Radio bajo o medio: 6-10px. Para CTAs tipo sello, usar formato pill.
- Sombras duras desplazadas: `4px 4px 0 #000`.
- Estados hover con desplazamiento pequeno o cambio de sombra.
- Iconos simples en botones de accion.

Tipos:

- Primario: fondo rosa, texto negro, borde negro.
- Secundario: fondo blanco, texto negro, borde negro.
- Oscuro: fondo negro, texto blanco, borde negro o blanco.

### Tarjetas

Las tarjetas deben recordar a pegatinas, flyers o carteles:

- Fondo blanco o blanco roto.
- Borde negro grueso.
- Sombra dura negra o rosa oscuro.
- Encabezados con etiqueta rosa o negra.
- Evitar tarjetas flotantes muy suaves con blur.

### Fondos

Fondos recomendados:

- Negro solido para hero o secciones de impacto.
- Blanco roto para lectura y formularios.
- Rosa claro para bandas secundarias.
- Patrones discretos inspirados en circulos, pegatinas o lineas radiales.

No abusar de degradados. Si se usan, que sean muy sutiles y subordinados al lenguaje grafico del logo.

### Secciones

La web debe tener ritmo de cartel:

- Bloques claros y oscuros alternados.
- Titulares grandes.
- Elementos circulares o semicirculares como guiño al logo.
- Separadores graficos con bordes negros, no lineas grises finas.

## 7. Tratamiento del hero

El hero debe ser el lugar donde mas se note la marca.

Direccion recomendada:

- Fondo negro o blanco con composicion de alto contraste.
- Logo real grande, visible en el primer viewport.
- H1 muy directo: "Charanga Malavida" o "Charanga en Zamora".
- Subtexto corto: fiestas, pasacalles, bodas y eventos.
- CTA primario rosa: "Contratar la charanga".
- CTA secundario blanco/negro: "Ver actuaciones".

El logo circular puede aparecer:

- Como protagonista centrado o lateral.
- Como sello grande parcialmente recortado.
- Como marca de agua grafica de fondo, siempre legible.

Evitar un hero demasiado elegante, minimalista o tipo SaaS. Malavida necesita presencia de cartel y energia de directo.

## 8. Tratamiento de imagenes, videos y mapa

### Imagenes

Las fotos deben integrarse con el lenguaje del logo:

- Marcos negros gruesos.
- Recortes tipo sticker.
- Etiquetas rosas o negras encima.
- Composiciones con fotos reales y detalles graficos.

Evitar filtros apagados, fotos muy oscuras o imagenes genericas de stock.

### Videos

Los embeds de YouTube pueden ir dentro de marcos graficos:

- Borde negro.
- Fondo blanco o rosa claro.
- Titulo en display condensada.
- Boton de play o etiqueta rosa.

El bloque media debe sentirse como cartel de concierto, no como una galeria neutra.

### Mapa

El mapa debe seguir la identidad:

- Marcadores circulares con el logo o el corazon.
- Popups con borde negro y CTA rosa.
- Leyenda simple.
- Limitar colores del mapa para que no compita con el logo.

Si Leaflet usa tiles externos, conviene elegir un estilo claro y sobrio. Los marcadores deben llevar la personalidad, no el mapa base.

## 9. Reglas de uso del logo

Recomendaciones:

- Usar el logo completo en hero, footer y lugares de marca.
- Mantener margen alrededor del circulo.
- No deformar el logo.
- No aplicar sombras blandas ni efectos 3D.
- No ponerlo sobre fondos con poco contraste.
- Si se recorta como recurso grafico, hacerlo intencionadamente y a gran escala.
- Para favicons o marcadores pequenos, usar solo el corazon si se prepara una version simplificada.

Versiones utiles a preparar:

- Logo completo circular.
- Logo horizontal o wordmark para header.
- Isotipo del corazon para favicon, mapa y botones pequenos.
- Version blanco/negro para fondos complejos.

## 10. Que evitar para no perder identidad

Evitar:

- Paletas corporativas frias tipo azul grisaceo.
- Degradados morados o azules dominantes.
- Beige, crema o marron como identidad principal.
- Tarjetas muy redondeadas y suaves estilo app financiera.
- Tipografia fina o demasiado elegante.
- Hero minimalista sin logo grande.
- Sombras difuminadas como lenguaje principal.
- Iconografia demasiado delicada.
- Fotos oscuras con blur que escondan la energia real.
- Demasiados colores nuevos que compitan con rosa, negro y blanco.

La marca ya tiene mucho caracter. La web debe ordenar ese caracter, no diluirlo.

### Uso recomendado de tokens

- `--mv-pink-soft`: secciones de apoyo.
- `--mv-shadow-hard`: tarjetas, botones, media y mapa.
- `--mv-radius-pill`: botones principales y badges.

## 12. Como adaptar el estilo actual de la web a esta nueva direccion

La web actual ya tiene buena estructura, pero la paleta verde/teal/coral no pertenece al logo. La adaptacion deberia centrarse en sustituir esa direccion por un sistema negro, blanco y rosa.

Cambios generales:

- Sustituir verdes y amarillos por negro, blanco y rosa Malavida.
- Cambiar sombras de color por sombras negras duras.
- Usar el logo real en `public/logo`.
- Dar mas peso a titulares condensados.
- Introducir recursos circulares inspirados en el sello del logo.
- Convertir botones y tarjetas en piezas tipo sticker/cartel.

## Aplicacion practica en la web actual

### Header

- Usar fondo blanco o negro.
- Cambiar el wordmark provisional por el logo real o una version horizontal.
- Navegacion en mayusculas o semimayusculas, con peso alto.
- CTA rosa con borde negro: "Contratar".
- En movil, menu simple con fondo blanco y bordes negros.

### Hero

- Fondo negro dominante.
- Logo circular real grande, como protagonista.
- H1 con tipografia condensada: "Charanga Malavida".
- Subtexto claro: "Charanga en Zamora para fiestas, pasacalles, bodas y eventos".
- CTA principal rosa con texto negro.
- CTA secundario blanco con borde negro.
- Posible recurso: logo parcialmente recortado como sello gigante.

### Seccion de historia

- Fondo blanco roto o rosa suave.
- Tarjetas con borde negro y sombra dura.
- Copys cercanos, sin tono corporativo.
- Usar detalles de mascota, circulos o etiquetas tipo pegatina.

### Proximas actuaciones

- Cada actuacion debe parecer un mini cartel:
  - Fecha grande.
  - Municipio destacado.
  - Borde negro.
  - Etiqueta rosa para tipo de evento.
- CTA: "Reservar una fecha" en rosa.

### Mapa

- Mantener mapa sobrio.
- Marcadores circulares con logo/isotipo.
- Popups blancos con borde negro.
- Fechas y pueblos en tipografia fuerte.
- Evitar que el mapa base tenga demasiados colores.

### Media

- Video principal dentro de marco negro.
- Fondo de seccion negro o rosa suave.
- Titulo grande y muy grafico.
- Spotify dentro de una tarjeta blanca con borde negro.

### Repertorio

- Presentarlo como lista de temazos o bloques de fiesta.
- Tarjetas con iconos simples, borde negro y acento rosa.
- Evitar tono academico. Debe sonar a directo y plaza.

### Contacto

- Formulario con campos blancos, borde negro y labels claros.
- Boton enviar rosa, grande y contundente.
- WhatsApp como accion secundaria o paralela.
- Incluir una frase directa: "Dinos fecha, pueblo y tipo de fiesta".

### Footer

- Fondo negro.
- Logo real visible.
- Enlaces en blanco.
- Acentos rosas para redes o contacto.
- Composicion compacta, con energia de cierre de cartel.

## Direccion final recomendada

La web debe sentirse como una mezcla de:

- Pegatina de charanga.
- Cartel de fiesta popular.
- Mascota cartoon.
- Web moderna, rapida y clara.

La clave es controlar la energia del logo: usar su contraste, su rosa y su lenguaje circular, pero con una estructura web limpia para que contacto, agenda, mapa y SEO sigan funcionando perfectamente.
